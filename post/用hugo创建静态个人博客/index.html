用hugo创建静态个人博客

Jan 2, 2018
2 min read

环境简介

系统：Windows10

管理工具：Git

托管地址：github，需要github账号

安装

点击下载

环境变量

点击查看教程

查看版本

C:\Hugo>hugo version
Hugo Static Site Generator v0.62.0-6608F155 windows/amd64 BuildDate: 2019-12-23T10:26:18Z`

创建网站

在当前文件夹下创建名为blog的文件夹

下载主题

点击下载主题

这里我选择了m10c这个主题，按照主页的教程下载即可。

注意，要把主题下载到themes文件夹下。

-t 参数用于指定主题。后面可以写死到配置文件中，这样就不用每次运行都输入了。

修改网站配置文件

配置文件说明

baseURL = "http://example.org/"
footnoteReturnLinkContents = "↩"
title = "与浪之间"           # 这是网页URL上方显示的东西
languageCode = "zh-hans"
theme = "m10c"              # 使用的样式
paginate = 10               # 首页每页显示的文章数

# 基本参数
[params]
    AuthorName = "王MT"           # 作者名称
    since = "2019"                # 站点创建时间 
    description = "hello world"   # 站点简介
    avatar = "http://wx3.sinaimg.cn/large/006ML2Boly1g4o56elnswj30w20w2thy.jpg"   # 头像
    showArchiveCount = true       # 是否在归档页显示文章的总数
    moreMeta = true               # 是否显示字数统计与阅读时间
    Subtitle = "Hugo is Absurdly Fast!"


# 社交链接 
[[params.social]]
    name = "github"
    url = "https://github.com/bobowa"   

[[params.social]]
    name = 'menu'
    url = "https://www.instagram.com/nickname"

baseUrl

修改后影响在本地查看网页。

建议在上传到github时手动覆盖。

avatar

头像采用微博作为图床

也可以使用本地文件，本地头像存放于themes\m10c\static文件夹下。但是需要注意尺寸和名称问题。

params.social

可以添加多个链接。

展示顺序按照配置文件顺序来，展示时居中排列，一排最多五个图标。

图标的形状和name属性相关。有预置的名字在themes\m10c\data\m10c\icons.json中。注意！并不是内置图片，而是用svg.path通过这里的数据渲染出来的。

关于更多的图片可以在iconfont下载

新建博客

C:\Hugo>hugo new post/first.md
C:\Hugo\blog\content\post\first.md created

会在content文件夹下新建post文件夹，并创建first.md文件。

生成静态内容

hugo --baseUrl="https://bobowa.github.io/" --buildDrafts

生成的public文件就是要上传的文件

注意！这里是两个短斜杠

创建托管仓库

在github上新建项目，仓库名字必须是用户名(全小写).github.io 对于我而言就是bobowa.github.io

最好不要选择创建readme文件，不然push时会有一点小麻烦。

上传

git init   # 在public文件夹内     
git add .  # 这里有一个点    
git commit -m "hugo blog first commit"    
git remote add origin https://github.com/bobowa/bobowa.github.io.git
git push -u origin master

查看博客

访问用户名.github.io即可

更新

重新生成public文件后 git push

错误记录

绑定baseUrl出错

协议是https而不是http

git出错

git add 添加所有文件的通配符是 点 不是星号
